Carpetas:
    1- bd: Carpeta que contiene los snipets o querys de la base de datos listos para ejecutar.

    2- clases: contiene todos los metodos crud.

    3- css: contiene todos los scripts css que serviran para dar estilo personalizado, en caso de ser util.

    4- archivos: contiene todos las imagenes que necesitemos agregar a nuestras vistas.

    5- archivos: contiene todas las imagenes de los productos agregados.

    6- js: contiene todos los script javaScript para cada modulo donde ejecutaran las funciones crud.

    7- librerias: contendra librerias para la creacion de pdf, jquery, alertify, bootstrap y select2.

    8- procesos: contiene archivos intermedios entre las vistas.

    9- vistas: contendra todas las vistas de cada uno de los modulos.
